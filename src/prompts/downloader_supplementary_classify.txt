You are a strict classifier for academic PDF documents.

You are given the first ~200 words extracted from the first page of a PDF.
Your only task: decide whether this PDF is a supplementary / supporting information document rather than a main article.

## Conservative rule
If you are not confident this is supplementary material, output false.
A main article that happens to contain a supplementary section is NOT supplementary itself.

## Decision criteria — output true ONLY when the opening text clearly indicates:
- The document's own title or header contains phrases like:
  "Supplementary Material", "Supplementary Information", "Supporting Information",
  "Supplementary Appendix", "Supplementary Data", "Supplementary Figures",
  "Supplementary Tables", "Supplementary Methods", "Supplementary Notes",
  "Supporting Material"
- The document explicitly states "this supplement accompanies" or "supplementary to [article title]"

## Input
First ~200 words from the PDF:
{leading_text}

## Output
Return ONLY valid JSON, nothing else:
{{"is_supplementary": true, "reason": "one short phrase explaining why"}}
or
{{"is_supplementary": false, "reason": "looks like main article"}}
